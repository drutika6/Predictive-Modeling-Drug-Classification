# Predictive Modeling for Drug Classification

## Overview
This project predicts the type of drug based on patient medical information using Machine Learning.

## Dataset
The dataset contains:
- Age
- Sex
- Blood Pressure (BP)
- Cholesterol
- Na_to_K ratio

Target:
- Drug Type

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
